It's a demo using:

- [Svelte 3](https://svelte.dev/)
- [Rollup](https://rollupjs.org/)
- [Postcss with Sugarss](https://github.com/postcss/sugarss)
- [modular-css](https://m-css.com/)


## Usage

```
$ git clone git@github.com:lagden/modular-css-demo.git demo
$ cd demo
$ npm i
$ npm run dev
```
